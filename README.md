# diyEggIncubator

## 사용한 부품
1. Arduino Uno + USB 통신케이블
2. 온습도센서(DHT11)
3. 1CH 릴레이 모듈
4. 전구(20W)
5. I2C 1602 LCD
6. 능동 부저
7. 초음파 가습기 모듈
8. 무탈피커넥터(I-22)
9. 220V 전원케이블
10. 9V DC 어댑터
11. 점퍼선
12. 마이크로 5핀 케이블
13. 온습도계




## 코드 라이브러리 버전
1. Serial 통신 LDDX0
2. oo
3. ff
